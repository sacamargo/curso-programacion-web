CODIGO DE PROGRAMACION

Español
En el contexto de la programación, el código es el conjunto de instrucciones que un desarrollador ordena ejecutar a un computador. Dicho código está estructurado según las guías correspondientes a un lenguaje de programación específico.
English
In the context of programming, code is the set of instructions that a developer orders a computer to execute. Said code is structured according to the guides corresponding to a specific programming language.

HTML

Español
El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos
English
Hypertext Markup Language (HTML) is the code used to structure and display a web page and its content. For example, your content could be paragraphs, a bulleted list, or images and data tables.

CSS

Español
CSS «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado
English
CSS "Cascading Style Sheets", is a graphic design language for defining and creating the presentation of a structured document written in a markup language.

JavaScript

Español
JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico.
English
JavaScript is an interpreted programming language, a dialect of the ECMAScript standard. It is defined as object-oriented, prototype-based, imperative, weakly-typed, and dynamic.
